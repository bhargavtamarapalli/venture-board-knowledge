# Lean_Canvas

# Purpose
Summarize a startup idea on a single page.

## When to Use
- Early-stage ideas
- MVP planning
- Investor discussions

## Inputs Required
- Problem
- Customer
- Solution
- Revenue
- Costs

## Evaluation Framework
1. Problem
2. Customer Segments
3. Unique Value Proposition
4. Solution
5. Channels
6. Revenue Streams
7. Cost Structure
8. Key Metrics
9. Unfair Advantage

## Checklist
- Clear customer?
- Pain validated?
- Differentiated?
- Monetization defined?
- MVP feasible?

## Questions to Ask
- What painful problem is solved?
- Who pays?
- Why now?
- Why choose this product?

## Metrics
PMF, CAC, LTV, Activation, Retention

## Red Flags
- No customer
- No revenue path
- Feature not business
- No differentiation

## Best Practices
Keep each block concise and evidence-based.

## Output Format
Return a completed Lean Canvas table followed by key assumptions.
