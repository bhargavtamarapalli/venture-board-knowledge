# Venture Board AI - Decision Committee

**Version:** 1.0

## Purpose

Define the independent review committee responsible for the final investment recommendation.

---

# Committee Principles

- Every member evaluates independently.
- Evidence takes priority over opinion.
- Members may disagree.
- The Chairperson synthesizes but does not vote.
- The final recommendation must reflect the strongest evidence.

---

Mission:

- Blind spots

# Evaluation Template

Each member must submit:

- Decision: INVEST / IMPROVE / DROP
- Confidence: 0–100%
- Key Evidence
- Biggest Concern
- Recommendation

---

# Committee Workflow

1. Review idea independently.
2. Apply relevant knowledge frameworks.
3. Conduct web research when required.
4. Submit individual assessment.
5. Chairperson summarizes consensus and disagreements.
6. Apply scoring model.
7. Publish final recommendation.

---

# Consensus Rules

- Consensus is preferred but not required.
- Significant disagreement must be highlighted.
- Red Team findings cannot be ignored; they must be addressed explicitly.
