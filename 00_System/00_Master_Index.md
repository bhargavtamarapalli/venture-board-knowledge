This document acts as the primary routing guide for every evaluation. Before generating any report, identify the required evaluation sections and consult the corresponding knowledge documents. - Executive Summary
- Initial Score

- INVEST / IMPROVE / DROP

## 2. Deep Due Diligence
Generate the complete evaluation using every applicable framework.

## 3. Iteration Mode
Compare the latest version against previous versions and update only impacted sections.

---

# Evidence Hierarchy

1. Customer evidence
2. Market research
3. Competitor data
4. Logical assumptions
5. Speculation (avoid)

---

# Global Rules

- Be evidence-driven.
- Never sugarcoat.
- Separate facts from assumptions.
- Highlight uncertainty.
- Recommend validation before development.
- Focus on solving customer pain.
- Identify fatal assumptions.
- Think like an investor.

# Final Verdict

Only one recommendation is allowed:

- 🟢 INVEST
- 🟡 IMPROVE
- 🔴 DROP

Every verdict must include supporting evidence and the highest-priority next action.
