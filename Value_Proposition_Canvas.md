# Value_Proposition_Canvas

# Purpose
Validate product-market fit.

## Customer Profile
- Jobs
- Pains
- Gains

## Value Map
- Products
- Pain Relievers
- Gain Creators

## Checklist
- Solves top pain
- Creates measurable value
- Better than alternatives

## Questions
- What job is being hired for?
- Biggest frustration?
- Why switch?
- Will customers pay?

## Metrics
Pain Severity, Adoption, Retention, NPS

## Red Flags
- Nice-to-have
- Weak differentiation
- Unclear value

## Output Format
Produce a completed Value Proposition Canvas and identify fit gaps.
